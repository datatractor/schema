# Enum: InstallerTypes




_This enumeration allows the `Extractor->installation` to specify which installer software is to be used for the installation of the `Extractor`._



URI: [InstallerTypes](InstallerTypes)

## Permissible Values

| Value | Meaning | Description |
| --- | --- | --- |
| pip | None | Install the Extractor using 'pip' |
| conda | None | Install the Extractor using 'conda' |
