# Enum: InstallerTypes




_This enumeration allows the `Extractor->installation` to specify which installer software is to be used for the installation of the `Extractor`._



URI: [InstallerTypes](InstallerTypes)

## Permissible Values

| Value | Description | Comments |
| --- | --- | --- |
| pip | Install the `Extractor` using 'pip'. Will process `packages` using `pip install`<br/>and `requirements` using `pip install -r`. |  |
| conda | Install the `Extractor` using 'conda'. | Note that installation using `conda` is not yet supported by the API. |
