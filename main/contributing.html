Contribution guidelines

Contributions to this project are welcome. We pledge to follow the Contributor Covenant Code of Conduct.

In particular, if you have any suggestions, technical queries, or a feature request related to the schemas, please do not hesitate to open an issue on GitHub. For general questions related to the MaRDA Metadata Extractors WG, please use the discussion board on GitHub.
